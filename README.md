# Lunaro Gate

A two-part system that gates DSLRBooth photobooth access at venues without payment integration. A cashier web app generates a 4-digit code per paid order; a booth Electron app lets the customer enter the code to unlock DSLRBooth and start a session.

See [`LUNARO_GATE_SPEC.md`](./LUNARO_GATE_SPEC.md) for the full system spec and [`CLAUDE.md`](./CLAUDE.md) for the day-to-day working context.
